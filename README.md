## 🎯 What Is This?

**Learn Blender** adds a simple panel to Blender's 3D View sidebar. Click any button – it searches YouTube for the **best, most up-to-date tutorials** on that topic.

**No more:**  
❌ Opening browser tabs  
❌ Guessing what to search  
❌ Finding outdated 2016 tutorials  
❌ Getting overwhelmed by 50M results  

**Just:**  
✅ One click → Perfect tutorials  
✅ Current year content automatically  
✅ Focused learning inside Blender

## 📥 Installation

**1. Download** the file from this repository  
   - Click the green **Code** button above  
   - Select **Download ZIP**  
   - Extract the `.py` file  

**2. Open Blender** → `Edit` → `Preferences` → `Add-ons`

**3. Click** `Install` → select the downloaded `.py` file

**4. Enable** – check the box next to "Learn Blender"

**5. Find it** – In 3D View, press `N` → click the **"Learn Blender" tab**

---

## 🖱️ How to Use

**Three simple steps:**

1. **Press `N`** in 3D View to open sidebar
2. **Click** the "Learn Blender" tab
3. **Click any button** – YouTube opens with perfect tutorials

**That's it.** No menus. No settings. No confusion.
