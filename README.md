## 📌 What is Learn Blender?

A free, open-source Blender add-on that adds a **"Learn Blender" panel** to the 3D View sidebar. Search YouTube, see real results with creator names, save your favorites, and track your progress through guided learning paths.

**No API keys. No setup. Just click and learn.**

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Real YouTube Search** | Shows actual video results with titles, creator names, duration, and views |
| 🎬 **Creator Names** | See who made the tutorial - Blender Guru, CG Fast Track, Grant Abbitt, etc. |
| 🔗 **Visit Channel** | One-click to go to the creator's YouTube channel |
| 📚 **Bookmarks** | Save tutorials, add notes, track status (Saved → In Progress → Completed) |
| 🎯 **Learning Paths** | 4 guided roadmaps: Beginner's Journey, Modeling Fundamentals, Materials & Textures, Animation Basics |
| 📊 **Progress Tracking** | Check off topics as you learn, see your completion percentage |
| 📜 **Search History** | Remembers your last 8 searches |
| 📅 **Current Year Filter** | Automatically shows tutorials from the current year |
| 🚀 **Blender 5.0 Ready** | Fully compatible with Blender 5.0 (also works with 4.x and 3.x) |

---

## 🖥️ How It Works

### Search Tab
- Type any topic or use quick buttons (Beginner, Modeling, Animation)
- See real YouTube results with:
  - Tutorial title
  - Channel name (Blender Guru, CG Fast Track, etc.)
  - Duration
  - View count
  - Upload date
- **Watch** button - opens the video
- **Visit Channel** button - goes to the creator's channel
- **Save** button - adds to bookmarks

### Bookmarks Tab
- All your saved tutorials in one place
- Change status: Saved → In Progress → Completed
- Add personal notes
- Quick watch and channel access
- Progress stats show how many you've completed

### Paths Tab
| Path | Topics |
|------|--------|
| **Beginner's Journey** | Interface & Navigation, Basic Objects, Simple Modeling, Materials, Lighting, Camera & Render |
| **Modeling Fundamentals** | Mesh Editing, Extrude Tools, Subdivision Surface, Hard Surface, Organic Modeling, Boolean Operations |
| **Materials & Textures** | Shader Nodes, Principled BSDF, Image Textures, UV Unwrapping, Procedural Textures, PBR Setup |
| **Animation Basics** | Keyframes, Timeline & Dopesheet, Graph Editor, Simple Character Animation, Camera Animation, Render Animation |

Check off topics as you learn, watch your progress bar grow.

## 📥 Installation

1. Download `learn_blender.py` from the [releases page](https://github.com/suhaildegrate-design/Learn-Blender/releases)
2. **Blender** → Edit → Preferences → Add-ons → Install
3. Select the downloaded file
4. Enable "Learn Blender"
5. In 3D View, press `N` → Click the "Learn Blender" tab
